## 第二章 行前準備 ES6 基本功
###解構復值

#####陣列解構
- 預設值
- 忽略元素
- 變數交換
- 剩餘部分
```
### 陣列解構 預設值
const nums = [1,2]
const [first,second third =9] = nums;

### 結果
first = 1
second =2
third = 9
```

##### 物件解構
- 預設值 ( z= 10)
- 指派新名稱 (x-px)
```
### 物件解構 預設值
const nums = {x:100,y:150}
const {x,y,z = 10} = nums;

### 結果
x = 100
y = 150
z = 10
```

#####  解構函式參數
```
function math(point){
    const{x,y} = point
    return Math.sqrt(x*x + y*y)
}

```

### 字串模板
- 插入表達式
- 多行字串

```
function hello(name){
    console.log(`Hello,${name} !`)
}
hello('john')

### 結果
Hello john !
```

### 箭頭函式
- 語法短
參數只有一個可省略（）
內容只有一行並回傳實可接為一行 （省略 return）
- 自動綁定
this 等於外面，不會跑到奇怪的層
```
const double = (x) =>{
    return x*2;
}
```
