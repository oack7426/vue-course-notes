###解構復值

#####陣列解構
- 預設值
- 忽略元素
- 變數交換
- 剩餘部分
```
### 陣列解構 預設值
const nums = [1,2]
const [first,second third =9] = nums;

### 結果
first = 1
second =2
third = 9
```

##### 物件解構
- 預設值 ( z= 10)
- 指派新名稱 (x-px)
```
### 物件解構 預設值
const nums = {x:100,y:150}
const {x,y,z = 10} = nums;

### 結果
x = 100
y = 150
z = 10
```
